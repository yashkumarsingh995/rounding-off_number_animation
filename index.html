Rounding-off

. 2 4 3 5 4 x 7 3

This is the number upto which we have to round-off this value

when (i+1)th number is greater than half of unity
We generally increment the value of the ith element by one

when (i+1)th number is lesser than half of unity
We generally do not increment the value of the ith element

when (i+1)th number is equal than half of unity
We generally increment the value of the ith element by one if ith value is odd and it will remain same if the ith value is even
